SublimeLinter-contrib-standard
-------------------------------
This linter plugin for SublimeLinter provides an interface to standard.

** IMPORTANT! **

Before this plugin will activate, you *must*
follow the installation instructions here:

https://github.com/Flet/SublimeLinter-contrib-standard

Want to automatically format your text? 
Install StandardFormat from package control!
(https://github.com/bcomnes/sublime-standard-format)
